Task 2 Submission

Task 2: Reconnaissance, Port Scanning & Network Traffic Analysis
Objective
The objective of this task was to understand the fundamentals of reconnaissance, network scanning, service enumeration, and network traffic analysis using industry-standard cybersecurity tools.

Tools Used
Nmap
Wireshark
Whois
Nslookup
Shodan
Google Dorking
Linux Terminal
Activities Performed
1. Reconnaissance
Passive Reconnaissance
Gathered publicly available information using Whois.
Performed DNS lookups using Nslookup.
Explored search engine reconnaissance through Google Dorking.
Analyzed publicly exposed services using Shodan.
Active Reconnaissance
Conducted host discovery using Ping Sweep.
Performed banner grabbing to identify running services and software versions.
2. Port & Service Scanning
Executed TCP SYN scans using Nmap.
Performed UDP port scans.
Identified open ports and associated services.
Enumerated service versions running on target systems.
3. Network Traffic Analysis
Captured network packets using Wireshark.
Analyzed protocols such as TCP, UDP, DNS, and HTTP.
Observed packet flow and communication patterns.

Learning Outcomes
Through this task, I gained hands-on experience in reconnaissance techniques, network scanning methodologies, and packet analysis. I learned how security professionals identify hosts, discover services, and analyze network traffic for security assessments.
